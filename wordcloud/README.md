# Word Cloud

## What this is

Word Cloud is a quick-fire facilitation tool for gathering language from a group. The facilitator sets a question, participants open the tool on their own devices and type a word or short phrase in response, and a live word cloud builds on the facilitator's screen in real time — sized by how often words appear.

It's useful any time you want to surface what words people reach for first: to open a session, to close one, to check what's top of mind before diving into a topic, or just to see whether the group is using the same language to talk about the same things.

---

## The views

**Facilitator view**
Set the question and share the QR code. The word cloud builds itself as responses come in — the more a word is submitted, the larger it appears. You can clear all responses at any time and run a fresh round.

**Participant view**
Participants scan the QR code, see the active question, and type their word or phrase. Submission is instant. They can submit and immediately see their contribution land in the cloud.

---

## How to use

Open `index.html` in a browser on the facilitator's device. Switch to the Facilitator tab to set your question, then display the QR code so participants can join from their own devices.

Participants scan the QR code and land on the response view — no accounts, no app downloads. As they submit, the word cloud updates live on the facilitator's screen.

The tool uses Firebase Firestore to sync responses in real time, so participants can be on any device and any network.
