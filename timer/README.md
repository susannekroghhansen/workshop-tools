# Timer

## What this is

A workshop countdown timer that syncs across every device in the room. The facilitator sets the time and controls start, pause, and reset. Participants see exactly the same countdown on their own screens, updated in real time.

It removes the awkward dance of pointing at a clock or shouting "five minutes left" across a room. Everyone is looking at the same thing, and the facilitator stays in control without having to stand next to the projector.

---

## The views

**Facilitator view**
Set the duration, then start, pause, or reset the timer as needed. The controls are large and easy to hit mid-session without breaking your focus. The countdown is displayed prominently so it's readable on a projected screen.

**Participant view**
Participants scan the QR code and see the live countdown on their own device. The timer updates in real time — if the facilitator pauses or resets, everyone sees it immediately. No interaction required from participants; it's read-only.

---

## How to use

Open `index.html` in a browser on the facilitator's device. Set the countdown duration using the input, then share the QR code so participants can follow along on their own devices.

The tool uses Firebase Firestore to sync the timer state in real time, so it works across devices on any network. Participants just scan and watch.
